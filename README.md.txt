# 🏢 Portal Corporativo (Intranet)

Projeto front-end de um portal corporativo inspirado em sistemas de intranet como Liferay.

---

## 🚀 Tecnologias
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- jQuery  

---

## 🎯 Funcionalidades
- Dashboard com cards informativos  
- Notícias internas dinâmicas  
- Sidebar de navegação  
- Interface responsiva  
- Interação com jQuery  

---

## 💡 Objetivo
Projeto para portfólio focado em interfaces corporativas e desenvolvimento front-end com Bootstrap e jQuery.
